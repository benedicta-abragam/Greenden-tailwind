# Greenden-tailwind

Greenden is a modern, responsive  website built using *Tailwind CSS*. It’s designed to showcase a collection of plants, making it perfect for plant stores, boutiques, or personal portfolios.

## Live Demo
[Check out the live site](https://benedicta-abragam.github.io/Greenden-tailwind/)

## Features
- Fully responsive design for desktop and mobile  
- Clean, elegant layout with sections for Best Sellers, About, Customer Reviews, and Newsletter  
- Built entirely with Tailwind CSS — easy to customize and extend  
- Ideal for showcasing products or creating a small online plant store
